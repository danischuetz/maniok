# Core Library Overview

The core library provides services to

- Parse `workspace.json` to `SzrWorkspace` objects
- Create `DiagramModel` tree from `SzrWorkspace`
- Create `DocumentModel`tree from `SzrWorkspace`
- Render `Markdown` to `html`
- Create `SvelteFlow` nodes and edges from a `DiagramModel` tree
- Create `LayoutTree` from `SvelteFlow` nodes and edges
- Layout a `LayoutTree`
